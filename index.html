Snake game Arduino

snake game : เป็นเกมที่ต้องกดปุ่มเพื่อบังคับทิศทางให้งูกินผลไม้ให้ครบทั้งหมด 10 ลูก และถ้าหากชนที่ขอบหรือตัวงูก็จะทำให้จบเกมทันที โดยที่แต่ละครั้งที่กินผลไม้ได้จะทำให้ความเร็วของตัวงูนั้นเพิ่มขึ้น เพื่อเพิ่มระดับความยาก โดยที่จะมีการจัดระดับทั้งหมด 4 ระดับ คือ
๐ กินได้ตั้งแต่ 1-3 ลูก คือ Biginner
๐ กินได้ตั้งแต่ 4-5 ลูก คือ Intermediate
๐ กินได้ตั้งแต่ 6-8 ลูก คือ Professional
๐ กินได้ตั้งแต่ 9 ลูกขึ้นไป คือ Master


การแสดงผลของเกม(output)
จะแสดงผลผ่านหน้าจอssd1306
ที่ต่อเข้ากับ arduino board

ส่วนของปุ่มบังคับทิศทาง
ในตัวเกมนั้นจะสามรถทำให้งูเปลี่ยน
ทิศทางได้โดยการกดปุ่มทั้งหมด 4 ปุ่ม คือ
๐ ปุ่มสีแดง คือ ด้านบน
๐ ปุ่มสีเขียว คือ ด้านขวา
๐ ปุ่มสีน้ำเงิน คือ ด้านล่าง
๐ ปุ่มสีเหลือง คือ ด้านซ้าน

ส่วน Setup ของตัวเกมจะติดต่อระหว่างบอร์ด Arduino กับ SSD ด้วยคำสั่ง Serial. Begin เพื่อกำหนดอัตราความเร็วในการรับส่ง หากระหว่างตัวบอร์ดเเละหน้าจอเกิดปัญหาขึ้นเเละไม่สามารถเชื่อมต่อกันได้จะขึ้นข้อความ Error ขึ้นมา เเละจะมี loop คำสั่ง INPUT_PULLUP ด้านล่างจะให้เป็นสัญญานของINPUT หลังจากนั้น run function ของ setupgame()

ButtonPress() จะทำการอ่านค่าจากการกดปุ่มเเละreturnค่าเป็นBoolean กลับไปนำไปใช้ในตอนเริ่มเกม
readdirection() รับทิศทางจากการกดปุ่ม

loop การทำงานของตัวเกม จะเเบ่งออกเป็น 3 state หลักๆคือ start(ก่อนเริ่มเกม) running(รันตัวเกม) เเละ gameover(จบเกม) เเต่ละ state ก็จะมีคำสั่งเเละ function ตามเเต่ละ state ของตัวเอง

movesnake() function การเคลื่อนไหวของงูเป็นการการหนดทิศทางตามเเนวเเกน X,Y

checkFruit() function การเก็บผลไม้หากเก็บได้ความยาวของเเละความเร็วของงูจะเพิ่มขึ้น
generateFruit() function การเกิดของผลไม้ จะสุ่มเลขเเละนำเลขมากำหนดตำเเหน่งx,y ของผลไม้
collision check() function เช็คการชนว่างูชนกำเเพงหรือชนหางหรือไม่เเละ return ค่า boolean กลับไป

show() function การจัดเเรงค์ตามคะเเนนที่ผู้เล่นได้รับเเบ่งเป็น 4 ระดับ
